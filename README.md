## Descripción  
Aplicación móvil para acceder a recursos educativos.  
El acceso a los recursos se realiza por medio de un **Login**, y si el usuario no está registrado, puede crear una cuenta mediante un formulario de **Registro**.  

---

## Funcionalidades principales  
- **Login / Registro de usuarios**  
- **Visualización de recursos**: título, descripción, tipo, imagen y url
- **Agregar más recursos**
- **Actualización de recursos**
- **Eliminación de recursos**

---

## Validaciones implementadas  
- No se permite que la descripción pase de 200 caracteres.
- No se permite que el título sea mayor a 50 caracteres.
- Las imagenes y links hacia los recursos, solo pueden ser urls.  

---

## Pantallas  
1. Login / Registro  
2. Listado de recursos
3. Agregar recurso  
4. Editar recurso
